deleteCharacterizationDefinition {ROhdsiWebApi}R Documentation

Delete Characterization id definition.

Description

Delete Characterization id definition.

Usage

deleteCharacterizationDefinition(characterizationId, baseUrl)

Arguments

characterizationId

An integer id representing the id that uniquely identifies a Characterization definition in a WebApi instance.

baseUrl

The base URL for the WebApi instance, for example: "http://server.org:80/WebAPI".

Details

Delete the Characterization definition from WebAPI for a given Characterization id

Value

None, unless error.

Examples

## Not run: 
deleteCharacterizationDefinition(characterizationId = 13242,
                                 baseUrl = "http://server.org:80/WebAPI")

## End(Not run)
